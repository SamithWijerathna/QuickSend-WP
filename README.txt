== Description ==
A plugin to list, search and select multiple files from your WordPress installation and transfer them via FTP/SFTP with real-time progress monitoring.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/quick-send-ftp-transfer` directory
2. Activate the plugin through the 'Plugins' screen
3. Configure connection settings under Tools > File Transfer

== Frequently Asked Questions ==
= What protocols are supported? =
Supports both FTP and SFTP connections

= What is the maximum file size? =
No inherent limit - uses chunked transfers
